# Wix Launch Checklist

Use with the master [playbook checklist](../../playbook/CHECKLIST-LAUNCH.md). This adds Wix-only steps.

---

## Before build

- [ ] Chrome logged into Wix (`~/.organized/bin/wix-do status`)
- [ ] Apple Events enabled (`~/.organized/scripts/chrome-enable-automation.sh`)
- [ ] CDP enabled for full automation (`~/.organized/scripts/chrome-cdp-start.sh --ensure`)
- [ ] `brief.json` filled from [CLIENT-BRIEF.md](CLIENT-BRIEF.md)
- [ ] Images copied to client `03-Assets/` folder

---

## Template & pages

- [ ] Site created (blank or chosen template)
- [ ] All agreed pages exist (Home, About, Contact, etc.)
- [ ] Page names match brief
- [ ] Header shows correct business name
- [ ] Mobile layout checked in Wix mobile view

---

## Content

- [ ] Logo uploaded to Wix Media
- [ ] Hero image set
- [ ] About text matches approved brief
- [ ] Services / menu / offerings accurate
- [ ] Contact info correct (phone, email, address, hours)
- [ ] Social links in footer or contact section

---

## Forms & contact

- [ ] Wix Form or contact method tested
- [ ] Submissions reach client inbox
- [ ] Thank-you message or redirect configured

---

## SEO (Wix dashboard)

- [ ] Site title and description set (Marketing → SEO)
- [ ] Each page has a unique title
- [ ] Favicon uploaded
- [ ] Google Business Profile linked (optional)

---

## Domain & publish

- [ ] Domain connected (client-owned or purchased)
- [ ] DNS verified (green check in Wix)
- [ ] Site **Published** (not just saved)
- [ ] HTTPS works at live URL
- [ ] Test on phone

---

## Handoff

- [ ] Client has Wix account access (owner or editor role)
- [ ] Short video or doc: how to edit text and swap images
- [ ] Explained Wix billing (if on their card)
- [ ] Archived `brief.json` and final screenshots
