# Which kit should I use?

Answer these questions in order.

## 1. Is this a freelancer / personal portfolio?

**Yes** → Use **[Portfolio Starter](../starters/portfolio/)**

- One scrolling page
- Calendly, social links, payment buttons
- GitHub Pages
- Good for: web designers, photographers, consultants selling themselves

**No** → Continue to question 2.

---

## 2. Does the client need to edit the site themselves?

**Yes** → Use **[Wix Kit](../starters/wix/)**

- Drag-and-drop editor for the client
- You build in Wix Studio; they maintain content later
- Good for: restaurants, salons, shops where staff update hours and photos

**No** → Continue to question 3.

---

## 3. Is it a standard small business brochure site?

**Yes** → Use **[HTML Business Starter](../starters/html/)**

- Home, About, Services, Contact
- Fast, free hosting on GitHub Pages
- You handle updates (or teach client to edit one config file)
- Good for: plumbers, contractors, local services, consultants

---

## Still not sure?

| Situation | Recommendation |
|-----------|----------------|
| Client wants online store with cart | Wix or Shopify — not these starters |
| Client wants a blog they write often | Wix or WordPress |
| Teen / beginner builder selling Wix sites | Portfolio starter for **their** marketing site; Wix for **client** deliverables |
| Client budget under $300, simple presence | HTML starter or Wix blank template |
| Client insists on "I want to click and edit" | Wix |
